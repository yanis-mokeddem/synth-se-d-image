Pour lancer le programme, il faut télécharger Processing (c'est un éxécutable) en version 2 (avec la 3 il y a des problèmes avec la kinect) et il faut aussi télécharger la librairie SimpleOpenNI qui sera utilisée par Processing.

Lien(Utile):

https://processing.org/download/
http://openni.ru/files/simpleopenni/index.html
https://code.google.com/archive/p/simple-openni/downloads

Nous avons codé sur Windows car nous avons rencontré plusieurs problèmes concernant l'installation de la librairie de SimpleOpenNI sur Linux.

Une fois Processing lancer, il suffit d'ouvrir le code source (sketch_171213a.pde) qui est dans notre dossier "sketch_171213a" en allant dans l'onglet "File" puis "Open...". Enfin une fois le code chargé dans la fenetre de Processing, on lance le code grace au bouton lecteur qui se trouve en haut a gauche.
